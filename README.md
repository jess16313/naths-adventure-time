## WELCOME TO A WORLD OF MYSTERY

This project is the product of insane aspirations for a birthday party. The idea is to take the logic and technology you may see at a commercial escape room game place, and repurpose it for a smaller, more intimate event. 

This repository relies on an external database hosted by Supabase. It provides players with a hub of information and the game master an easy way to update this information simultaneously across devices.
